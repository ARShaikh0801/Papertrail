# LinkedIn Post — NoteFlow Project

---

🚀 **Excited to share my latest full-stack project — NoteFlow!** 📝

A clean, modern Notes app built completely from scratch using **React 19 + Vite** on the frontend and **Django REST Framework + MongoDB Atlas** on the backend.

### 🔑 Key Features:
✅ JWT-based user authentication (Register & Login)
✅ Full CRUD — Create, Edit, Delete notes
✅ Pin important notes to the top
✅ Interactive checklists with checkable items
✅ Dark/Light mode toggle with persistent preference
✅ Fully responsive across mobile, tablet & desktop
✅ Quick-create inline form for rapid note-taking

### 💡 What I learned building this:
• Building a **custom JWT auth system** from scratch with PyJWT — no third-party auth libraries
• Using **MongoEngine** with Django (no SQL, no migrations — just document models!)
• Designing a **clean component architecture** in React with reusable modals, cards, and forms
• Implementing a **full CSS theming system** using CSS variables for dark mode
• Writing responsive layouts that work beautifully on all screen sizes

### 🛠️ Tech Stack:
**Frontend:** React 19 · Vite 7 · React Router · Axios
**Backend:** Django 4.2 · Django REST Framework · MongoEngine
**Database:** MongoDB Atlas
**Auth:** Custom JWT (PyJWT)
**Styling:** Vanilla CSS with CSS Variables

This was an incredible learning experience — from setting up CORS headers to building JWT interceptors in Axios, every piece taught me something new. 💪
